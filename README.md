# Projeto-ONG

# 📋 Contexto do Projeto Original

### Descrição
O projeto original foi a criação de um site integrado de doações, buscando fortalecer a cultura de solidariedade e responsabilidade social. Já que a ausência de plataformas digitais eficientes e seguras para doações limita o impacto social de ONGs que atuam na área educacional. O projeto inclui telas do site, como tela de pagamento, página de doação, página de FAQ e visualização de campanhas.
