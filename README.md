# Projeto-ONG

# 📋 Contexto do Projeto Original

### Descrição
O projeto original foi a criação de um site integrado de doações, buscando fortalecer a cultura de solidariedade e responsabilidade social. Já que a ausência de plataformas digitais eficientes e seguras para doações limita o impacto social de ONGs que atuam na área educacional. O projeto inclui área de pagamento, área de doação, área de "Fale conosco" e visualização de campanhas.

### Funcionalidades Existentes
- Área de pagamento;
- Tela de doações realizadas;
- Botões dinâmicos;
- Seção de "Fale conosco".

### Tecnologias Utilizadas
- Frontend: CSS;
- Backend: HTML, PHP;
- Banco de dados: MySQL.
